Connect+ 
 Brief Description

Connect+ is a social networking mobile application developed for college students using Flutter and Dart. The application enables students to connect, communicate, create posts, and interact within the university community.

 Technology Stack and Tools Used

- Flutter
- Dart
- Firebase
- Cloudniary
- Android Studio
- VS Code
- GitHub
- JavaScript

 Features and Functionalities Implemented

- User Authentication
- Social Media Feed
- Create and Share Posts
- User Profile Management
- Edit Profile Feature
- Responsive UI
- Splash Screen
- Firebase Integration

---

  Installation / Execution Steps

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open project in VS Code or Android Studio

3. Install dependencies

```bash
flutter pub get
```

4. Run the project

```bash
flutter run
```
